Tic-Tac-Toe

This is a simple Tic-Tac-Toe game built using HTML, CSS, and JavaScript. It is a two-player game where users take turns marking spaces in a 3×3 grid with either an "X" or an "O". The objective is to get three marks in a row (horizontally, vertically, or diagonally) before the opponent
Features: ✅ Interactive UI: Clickable grid cells for making moves. ✅ Winning Logic: Detects when a player wins or if the game ends in a draw. ✅ Game Reset Option: Resets the board to play again. ✅ Smooth Transitions & UI Effects: Uses animations for better user experience. ✅ Responsive Design: Works on different screen sizes

Stopwatch

This Stopwatch project is a simple yet functional web-based timer built using HTML, CSS, and JavaScript. It allows users to start, pause, and reset the time with precision. The stopwatch displays elapsed time in hours, minutes, seconds, and milliseconds.
Features: ✅ Start, Stop & Reset Controls – Users can start, pause, and reset the stopwatch with buttons. ✅ Real-Time Timer – Updates every millisecond for accuracy. ✅ Lap Feature (Optional) – Users can record lap times while the stopwatch is running. ✅ Smooth UI Animations – Button hover effects and transitions for a better experience. ✅ Responsive Design – Works well on desktops, tablets, and mobile devices.
